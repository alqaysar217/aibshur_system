# **App Name**: Absher Delivery

## Core Features:

- User & Profile Management: Allows clients, drivers, and admins to create, update, and view their profiles, including role-specific details and account status.
- Store & Product Catalog: Browse stores by city, categories, and filters. View product details, including variants, pricing, and imagery.
- Order Management & Tracking: Clients can place orders, select payment methods, and track their order status from pending to delivered. Drivers can manage assigned orders and update delivery status.
- Driver Operations & Geo-indexing: Drivers can toggle online status, manage their vehicle details, and access geo-indexed store and client locations for efficient navigation.
- Financial Transactions & Wallet: Users can view transaction history, top-up wallets, and manage withdrawals. Integrates payment methods as specified.
- System Configuration & Promotions: Admins can manage system settings, VIP packages, coupons, loyalty points, app information, and ads banners.
- Personalized Recommendations Tool: Generative AI tool suggesting relevant stores, products, or promotions to clients based on their order history and browsing behavior.

## Style Guidelines:

- Color scheme: Light mode. Aim for a sense of reliability and positivity, reflecting the 'good news' connotation of Absher.
- Primary color: Deep, professional blue (#1F3C88) for conveying trust, efficiency, and reliability.
- Background color: Very light blue-grey (#F0F4FA) for a clean, expansive, and focused interface, providing subtle continuity with the primary hue.
- Accent color: Clear, medium-bright blue (#5EAFD6) to highlight calls to action, active states, and important information, ensuring contrast without being overly vibrant.
- Primary font: 'PT Sans' (humanist sans-serif) for all text. Its modern yet warm aesthetic ensures excellent readability and universal appeal, especially considering the Arabic RTL language requirement.
- Utilize a consistent set of clean, vector-based icons that are universally recognizable for delivery actions, store categories, and user functionalities, adaptable for Arabic RTL presentation.
- Emphasize clear hierarchy and spatial organization with a card-based layout for stores and products, intuitive navigation for clients and drivers, and distinct dashboard elements for admin functionalities.
- Implement subtle and purposeful animations for state changes, such as order status updates, loading screens, and menu transitions, providing immediate visual feedback and a polished user experience.
